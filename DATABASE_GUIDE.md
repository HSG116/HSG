## 📊 جداول قاعدة البيانات

### جدول `messages`
| العمود | النوع | الوصف |
|--------|------|-------|
| id | BIGSERIAL | المعرف الفريد |
| name | TEXT | اسم المرسل |
| email | TEXT | البريد الإلكتروني |
| project_type | TEXT | نوع المشروع |
| message | TEXT | نص الرسالة |
| created_at | TIMESTAMPTZ | تاريخ الإرسال |

### جدول `site_projects`
| العمود | النوع | الوصف |
|--------|------|-------|
| id | BIGSERIAL | المعرف الفريد |
| name_ar | TEXT | اسم المشروع بالعربية |
| name_en | TEXT | اسم المشروع بالإنجليزية |
| description_ar | TEXT | الوصف بالعربية |
| description_en | TEXT | الوصف بالإنجليزية |
| category | TEXT | الفئة (web, ai, islamic, tools) |
| image_url | TEXT | رابط الصورة |
| url | TEXT | رابط المشروع |
| created_at | TIMESTAMPTZ | تاريخ الإضافة |

## 🔐 الأمان

### Row Level Security (RLS)
تم تفعيل RLS على جميع الجداول مع السياسات التالية:

**للرسائل:**
- ✅ الجميع يمكنهم إضافة رسائل
- ✅ الجميع يمكنهم قراءة الرسائل
- ✅ الجميع يمكنهم حذف الرسائل

**للمشاريع:**
- ✅ الجميع يمكنهم قراءة المشاريع
- ✅ الجميع يمكنهم إضافة/تعديل/حذف المشاريع

> ⚠️ **ملاحظة:** يمكنك تقييد الصلاحيات لاحقاً لتكون متاحة للمشرفين فقط

## 📦 إضافة مشاريع جديدة

### من لوحة تحكم Supabase
1. اذهب إلى **Table Editor**
2. اختر جدول `site_projects`
3. اضغط **Insert** → **Insert row**
4. املأ البيانات:
   - `name_ar`: اسم المشروع بالعربية
   - `category`: الفئة (web, ai, islamic, tools)
   - `description_ar`: وصف المشروع
   - `image_url`: رابط الصورة
   - `url`: رابط المشروع
5. اضغط **Save**

ستظهر المشاريع الجديدة تلقائياً في الموقع!
